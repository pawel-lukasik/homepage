HTML: znaczniki, linki, atrybuty, nawigacja

  • Znacznik: <znacznik atrybut=”wartość”> treść </znacznik>
  • <br>, <strong>, <em>
  • Raczej nie używamy już <i>, <u>, <b>
  • Linki: <a href="..." target="..."> opis łącza </a>
  • Nawigacja wewnątrz jednej strony: <element id="moj-identyfikator">, później link do tego <a href="#moj-identyfikator" > opis łącza </a>
  • Nawigacja pomiędzy podstronami: znacznik <nav> ...link wewnętrzny... </nav>, można to umieścić w <header>
  • Obrazek: <img src="..." alt="...">

CSS część 2

  • atrybut stylu w linii html nazywamy inline np.: <znacznik style=”...”> treść </znacznik>
  • selektory: #id, .class, element
  • reguła !important (najważniejsze stylowanie)
  • hierarchia stylowania: 1. !important 2. style="" (inline) 3. #name 4. .name 5. element
  • w html: class="name", w css: .name
  • w html: id="name", w css: #name
  • stylujemy po klasach (bez zagnieżdżeń!)
  • nazywamy klasy po angielsku zgodnie z przeznaczeniem
  • nie używamy !important i stylów inline'owych jeśli nie jest to konieczne
